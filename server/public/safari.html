Safari Mobile Setup

Follow these steps to use Sharely on your iPhone Safari browser.

1 Save the "Sharely" Bookmark

Copy the code below, then bookmark THIS page in Safari. Then edit that bookmark and replace the URL with this code.

javascript:(function(){const k=prompt('Enter Sharely API Key:');if(!k)return;const u=window.location.origin;fetch('API_URL/api/config',{headers:{'X-API-Key':k}}).then(r=>r.json()).then(d=>{const s=d.services.find(i=>window.location.hostname.includes(i.domain.replace(/^\./,'')));if(!s){alert('Service not found for this domain');return;}const acc=s.accounts[0];if(!acc){alert('No accounts found');return;}acc.cookies.forEach(c=>{document.cookie=`${c.name}=${c.value}; domain=${c.domain}; path=${c.path}; expires=Tue, 19 Jan 2038 03:14:07 GMT; secure`;});alert('Cookies injected! Refreshing...');location.reload();}).catch(e=>alert('Error: '+e));})();
2 How to use
  • Go to netflix.com (or any service) in Safari.
  • Tap the Bookmarks icon.
  • Tap the Sharely bookmark you created.
  • Enter your API Key when prompted.
  • Page will refresh and you'll be logged in!
