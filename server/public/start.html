How it works

Up and running in minutes

1

Install the Sharely extension

Download the extension file and load it into Chrome or Edge. It only takes 30 seconds — no account required at this step.

⬇ Download .zip file How to load a .zip →
1. Unzip the file  ·  2. Go to chrome://extensions  ·  3. Enable Developer mode  ·  4. Click "Load unpacked"  ·  5. Select the unzipped folder
2

Configure the server

Open the extension by clicking the Sharely icon in Chrome. Click the ⚙ Settings icon at the bottom and enter the server URL you received.

3

Enter your email and access token

After setting the server, a login screen will appear. Enter the email address you used to purchase access and paste in your access token. Then click Get Access.

One device per account

Your account is bound to the device you first log in with. Contact support if you need to switch devices.

⏰

Access duration

Your access is valid for the duration shown in the extension footer. You'll receive an email reminder before it expires.

✉️

Magic link login

After registering, you log in by email only — no password needed. Just open the extension and enter your email.
